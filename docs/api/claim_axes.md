# Claim Axes

`claim_axes.py` defines the shared evidence, claim-status, admission, and
freshness vocabulary used by finding ingest and recall rendering. It keeps the
admission boundary and the read-side presentation lattice on the same wire
values, including the invariant that falsified evidence cannot be labelled
reference-validated.

The machine-readable cross-repository schema is generated by
`remanentia claim-schema` into `docs/schema/remanentia_claim_axes.schema.json`.
