# Remanentia

Persistent neural memory for AI agents.

A spiking neural network with STDP that accumulates experience across sessions. Not a vector database — a living network whose synaptic weights encode associative relationships through spike-timing dependent plasticity.

## Status

Under active development. Internal codebase in [04_ARCANE_SAPIENCE](https://github.com/anulum). Export to this repository when ready.
