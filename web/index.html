System split

Two responsibilities, one control loop.

REMANENTIA

Remembers

Indexes project files, session logs, handovers, public documents, compiled facts, and refreshable vector corpora.

  • Scheduled vector refresh worker
  • BM25, embeddings, reranking, compiled recall
  • Allowlisted public vector search
DIRECTOR-CLASS-AI

Verifies

Routes tasks, checks evidence, guards answers, and enforces the private/public boundary before output leaves the system.

  • Factuality guard and response verifier
  • Memory-backed decision context
  • Policy-controlled public output

Control loop

From request to evidence-gated answer.

1. RequestAgent, application, or API call
2. RouteDIRECTOR-CLASS-AI selects policy and path
3. RetrieveREMANENTIA returns memory and vector evidence
4. VerifyFactuality and public-safety checks run
5. AnswerBounded, cited, and policy-compliant output

Public safety

Private memory does not mean public exposure.

Internal logs and handovers can improve operational recall, but public APIs must expose only allowlisted corpora, redacted snippets, and evidence that passed the public-output policy. Raw vector results are evidence for the control layer, not automatic public text.

Private operational recordsInternal by default
Public vector resultsAllowlisted + redacted
Legacy wordingBlocked or rewritten
Final answersEvidence-gated
